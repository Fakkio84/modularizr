modularizr

A JS function that allow creations of modularize function that share public, protected and private properties and methods.

.make(modules)

Description

Makes an object build on an array of modules. All modules must be registered before they can be used to make an object with this method.

Parameters

Name Type Description
modules array An array of strings that contains the modules names from which build the object.

Returns

qqwqw

Example

modularizr.make([
	"module1",
	"module2"
]);
