Modularizr

A JS function that allow creations of modularize function that share public, protected and private properties and methods.

MODULARIZR

Description

Modularizr add a single variable to the global object (window in browsers). Implements three utility function to create objects that can take advantage of private, protected and public properties and methods.

.register(name, module)

Description

Registers a module for use it to build singletons or classes.

Parameters

Name Type Default Description
name string required The module's name.
module function(publicScope, protectedScope, parameters): {publicScope, protectedScope} required The module's body.

Returns boolean

True if all went ok.

Example

MODULARIZR.register('module', function(publicScope, protectedScope, parameters){
	'use strict';
	
	publicScope.logArgs = function(){
		console.log(parameters);
	};

	return {
		publicScope: publicScope,
		protectedScope: protectedScope
	};
});

.singleton(modules[, parameters])

Description

Builds a singleton from the modules passing them an optional array of parameters.

Parameters

Name Type Default Description
modules string[] required An array of strings that contains the modules names from which build the object.
[parameters] *[] undefined An array of parameters that will be passed to each modules.

Returns object

The resulting object built from the modules application.

Example

var mySingleton = modularizr.singleton([
	"module1",
	"module2"
], [
	1,
	"one",
	{key: "value"}
]);

.class(modules)

Description

Builds a "class" function from the modules

Parameters

Name Type Default Description
modules string[] required An array of module names from which to build the "class" function.

Returns function(...*):classFunction

The resulting "class" function built from the modules application. It takes an arbitrary number of arguments and passes them to the modules.

Example

var myClass = modularizr.singleton([
	"module1",
	"module2"
]);
var myObject = myClass(1, "one", {key: "value"});
